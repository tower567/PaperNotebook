# 📒 PaperNotebook

一个简单的仓库，用来记录我的论文阅读笔记。

## 📂 目录结构
- **Multi-turn-RL/**
  - **ARPO/**
    - `ARPO.md` ：论文阅读笔记（Markdown）
    - `ARPO.pdf`：论文原件
    - `code/`   ：相关代码（复现、实验或demo）

- 每篇论文单独一个子目录，名字 = 论文简称。
- 子目录下包含：
  - 一份 `.md` 笔记（核心内容、总结、实验要点）
  - 一份 `.pdf` 原文
  - 可选的 `code/` 文件夹存放相关实现
